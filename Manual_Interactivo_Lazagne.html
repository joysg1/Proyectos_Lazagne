¿Qué es LaZagne?

LaZagne es una herramienta de código abierto utilizada para recuperar contraseñas almacenadas en un sistema local. Es capaz de extraer contraseñas de numerosas aplicaciones, navegadores web y sistemas de almacenamiento de credenciales.

Características principales

Multiplataforma

Compatible con Windows, Linux y macOS, lo que permite su uso en diferentes entornos.

Múltiples módulos

Incluye módulos para navegadores, aplicaciones de correo, FTP, bases de datos y más.

Fácil de usar

Interfaz de línea de comandos intuitiva con opciones claras y documentadas.

Código abierto

Desarrollado en Python y disponible en GitHub para su revisión y contribución.

Casos de uso legítimos

  • Recuperar contraseñas olvidadas en sistemas propios
  • Auditorías de seguridad autorizadas
  • Recuperación de datos después de fallos del sistema
  • Pruebas de penetración con autorización

Instalación de LaZagne

Requisitos

LaZagne requiere Python 2.7 o Python 3.x para funcionar. Algunos módulos pueden requerir bibliotecas adicionales.

Método 1: Clonar desde GitHub

git clone https://github.com/AlessandroZ/LaZagne.git
cd LaZagne
pip install -r requirements.txt

Método 2: Descargar binarios precompilados

Para usuarios de Windows, existen binarios precompilados disponibles en las releases de GitHub.

Verificación de la instalación

Para verificar que LaZagne se ha instalado correctamente, ejecuta:

python laZagne.py --help
Nota: En sistemas Linux/macOS, es posible que necesites usar python3 en lugar de python.

Uso Básico de LaZagne

Sintaxis general

python laZagne.py [opciones] [módulos]

Opciones principales

-h, --help # Muestra este mensaje de ayuda
--version # Muestra la versión del programa
-v, --verbose # Modo verbose (mostrar errores)
-quiet # Modo silencioso (solo mostrar contraseñas encontradas)
-oN # Guardar resultados en un archivo de texto
-oJ # Guardar resultados en un archivo JSON
-oA # Guardar resultados en todos los formatos disponibles
-all # Ejecutar todos los módulos disponibles

Ejecutar módulos específicos

Puedes ejecutar módulos específicos usando:

python laZagne.py [módulo]

Por ejemplo, para ejecutar solo el módulo de Chrome:

python laZagne.py browsers -chrome

Listar módulos disponibles

Para ver todos los módulos disponibles:

python laZagne.py --help-modules

Ejemplos de Uso

Ejemplo 1: Recuperar todas las contraseñas

Ejecutar todos los módulos disponibles:

python laZagne.py -all

Ejemplo 2: Recuperar contraseñas de navegadores

Ejecutar solo los módulos de navegadores:

python laZagne.py browsers

Ejemplo 3: Recuperar contraseñas de Chrome y guardar resultados

Ejecutar solo el módulo de Chrome y guardar los resultados en un archivo:

python laZagne.py browsers -chrome -oN chrome_passwords.txt

Ejemplo 4: Recuperar contraseñas WiFi

Ejecutar el módulo de WiFi:

python laZagne.py wifi

Ejemplo 5: Modo silencioso

Ejecutar en modo silencioso (solo mostrar contraseñas encontradas):

python laZagne.py -all -quiet

Ejemplo 6: Guardar en múltiples formatos

Ejecutar todos los módulos y guardar resultados en texto y JSON:

python laZagne.py -all -oA resultados
